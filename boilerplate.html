What Is It?

An open source tool for developing UI components in isolation.

Why Use It?

  • Gives you a bird's eye view of your component library
  • Demo all use cases without having to manipulate data (even hard to reach use cases

How To Use It?

  • Options for quick start or manual install with most JS frameworks/libraries
  • Install as a project dependency
  • Set up your story files
  • Deploy storybook as a static site to share with your whole team

Config

import { configure } from '@storybook/react'; import '../src/App.css'; function loadStories() { require('../stories/index.js'); } configure(loadStories, module);

Add Ons

There are a variety of plugins to add functionality to Storybook UI

  • Knobs edit component props dynamically within the Storybook UI
  • Actions display data received by event handlers
  • Storyshots works with Jest to take snapshots of your stories
  • Notes add detailed components notes using markdown
  • Accessibility: test component compliance with web accessibility standards
  • COMING SOON generate styleguide using markdown
